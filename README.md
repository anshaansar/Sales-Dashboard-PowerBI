# Sales-Dashboard-PowerBI

# Project Overview
This project is an interactive Sales Dashboard created using Microsoft Power BI to analyze sales performance and provide business insights.

# Files
- sales_dashboard.pbix – Power BI report
- SalesData.xlsx – Dataset used in the report
- Screenshot.png – Dashboard preview

# Dashboard Features
- Total Sales
- Total Profit
- Sales by Region
- Monthly Sales Trend
- Category-wise Analysis
- Interactive Filters (Slicers)

# Tools Used
- Microsoft Power BI
- Microsoft Excel
